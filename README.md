# 🎮 Tic Tac Toe — Where Strategy Meets Simplicity!

> **A sleek, interactive, and fully responsive Tic Tac Toe game built with HTML, CSS & JavaScript.**
> Challenge a friend, outsmart your opponent, and relive the timeless classic with a modern UI.

---

## 🌟 Features

* 🎯 Classic **2-Player Gameplay**
* ❌⭕ Automatic Turn Switching (X & O)
* 🏆 Instant Winner Detection
* 🤝 Draw Detection
* 🔄 One-Click Reset
* 🆕 Start a New Game Anytime
* ✨ Smooth Hover Animations
* 🎨 Minimal & Modern UI
* ⚡ Built with Vanilla JavaScript — No Frameworks

## 🎮 How to Play

1. Player **O** makes the first move.
2. Players alternate turns placing **X** and **O**.
3. Match **3 symbols** in a row, column, or diagonal to win.
4. If all 9 cells are filled without a winner, the game ends in a **Draw**.
5. Click **Reset** or **New Game** to play again.
